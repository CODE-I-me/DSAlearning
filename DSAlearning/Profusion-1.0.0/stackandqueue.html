Stacks and Queues in Data Structures

What is a Stack?

A stack is a linear data structure that follows the LIFO (Last In First Out) principle. The last element inserted is the first one to be removed.

Basic Operations on Stack

  • push() â€“ Adds an element to the top of the stack.
  • pop() â€“ Removes the topmost element.
  • peek() / top() â€“ Returns the topmost element without removing it.
  • isEmpty() â€“ Checks if the stack is empty.

Java Stack Example

import java.util.Stack;

public class StackDemo {
    public static void main(String[] args) {
        Stack<Integer> stack = new Stack<>();
        stack.push(10);
        stack.push(20);
        stack.push(30);
        System.out.println("Top element: " + stack.peek());
        stack.pop();
        System.out.println("After pop, top: " + stack.peek());
    }
}

What is a Queue?

A queue is a linear data structure that follows the FIFO (First In First Out) principle. The first element inserted is the first one to be removed.

Basic Operations on Queue

  • enqueue() â€“ Adds an element to the end of the queue.
  • dequeue() â€“ Removes an element from the front of the queue.
  • front() / peek() â€“ Returns the front element.
  • isEmpty() â€“ Checks if the queue is empty.

Java Queue Example (Using LinkedList)

import java.util.LinkedList;
import java.util.Queue;

public class QueueDemo {
    public static void main(String[] args) {
        Queue<Integer> queue = new LinkedList<>();
        queue.add(10);
        queue.add(20);
        queue.add(30);
        System.out.println("Front: " + queue.peek());
        queue.remove();
        System.out.println("After dequeue, front: " + queue.peek());
    }
}

Applications

  • Stacks: Expression evaluation, Undo functionality, Recursion handling
  • Queues: Job scheduling, Call center systems, Data buffering
